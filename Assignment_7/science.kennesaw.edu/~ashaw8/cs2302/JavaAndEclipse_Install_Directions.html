Java, Eclipse and Android SDK Installation
(scroll down for Mac and Linux directions)

On Windows:
  1. Install the Java SE Development Kit (JDK):
    1. Go to the following site: Java SE Development Kit 7 Downloads.
    2. Select your platform, then click on continue and follow the instructions to download and install.
  2. Download the Android SDK (includes Eclipse in the Package):
    1. Go to the following site: Get the Android SDK.
    2. Click on "Download the SDK" (ADT Bundle for Windows).
    3. Choose your platform and then "Download the SDK ADT Bundle for Windows".
    4. Unzip your "adt-bundle-windows..." folder and place it somewhere OUTSIDE OF YOUR "Downloads" folder (wherever you want).
  3. Install Eclipse and the Android SDK:
    1. Open your "adt-bundle-windows..." folder and then open your "eclipse" folder inside of that, and right-click on the first "Eclipse" icon you see (the purple icon) and rename it "Eclipse with Android", and if you see a ".exe" in the name, be careful to leave it there. If you don't see the ".exe" before you rename it, don't worry about it.
    2. Again, right-click on your "Eclipse with Android" icon and select either "Pin to Start Menu", "Pin to Taskbar" or "Create shortcut", depending on where you want to place your icon to start Eclipse.
      • "Pin to Start Menu" will put it in your Start Menu.
      • "Pin to Taskbar" will put it on your Taskbar.
      • "Create shortcut" will create an "Eclipse with Android" shortcut icon that you can drag to your Desktop.
    3. Note: You can have more than one version of Eclipse running on your system at a time without a problem. If you previously installed a different version of Eclipse, this will add a new one with its own workspace, and you can always choose which one to run.
    4. Click on the new icon you have created to start Eclipse. You may be asked for permission to run it (give it permission) and then you will be asked to "Select a workspace" folder, but the default location is not an easy location to get to later. I recomend that you either put it in your "Documents" folder (or "My Documents" for XP) or on your "Desktop". To do this, just put the words "Documents\" or "Desktop\" (or "My Documents\" for XP) in front of the word "workspace" in the dialog shown.
    5. The next screen will give you links to a set of Android Tutorials. Our class will not work with Android Apps at first. We will cover that material somewhat later. So you can skip those tutorials and just clicking on the "X" on the "Android IDE" tab.
    6. Now Eclipse is installed and ready to run Standard Java projects and Android projects.
    7. Note: Initially in this class, you will only be creating standard Java projects.

On Mac OS X
Java is already installed.
  1. Download the Android SDK (includes Eclipse in the Package):
    1. Go to the following site: Get the Android SDK.
    2. Click on "Download For Other Platforms" link near the bottom.
    3. Download the "Mac OS X 64-bit" Package option.
    4. Once downloaded, unzip your "adt-bundle-mac..." folder.
  2. Install Eclipse and the Android SDK:
    1. Open your "adt-bundle-mac..." folder and then open your "eclipse" folder inside of that, and then your "Eclipse.app" folder, then your "Content" folder, then your "MacOS" folder, then run "eclipse".
    2. You will be asked to "Select a workspace" folder, and choose a location for your workspace folder that will be easy for you to find later.
    3. Note: You can have more than one version of Eclipse running on your system at a time without a problem. If you previously installed a different version of Eclipse, this will add a new one with its own workspace, and you can always choose which one to run.
    4. The next screen will give you links to a set of Android Tutorials. Our class will not work with Android Apps at first. We will cover that material somewhat later. So you can skip those tutorials and just close the "Android IDE" tab.
    5. Now Eclipse is installed and ready to run Standard Java projects and Android projects.
    6. Note: Initially in this class, you will only be creating standard Java projects.

On Linux
  1. Install the Java SE Development Kit (JDK) if it is not already installed on your system:
    1. Go to the following site: Java SE Development Kit 7 Downloads
    2. Select your platform, then click on continue and follow the instructions to download and install.
  2. Download the Android SDK (includes Eclipse in the Package):
    1. Go to the following site: Get the Android SDK.
    2. Click on "Download For Other Platforms" link near the bottom.
    3. Download the "Linux 32-bit" or the "Linux 64-bit" Package option.
    4. Once downloaded, unzip your "adt-bundle-linux..." folder.
  3. Install Eclipse and the Android SDK:
    1. Open your "adt-bundle-linux..." folder and then open your "eclipse" folder inside of that, and then run "eclipse".
    2. You will be asked to "Select a workspace" folder, and choose a location for your workspace folder that will be easy for you to find later.
    3. Note: You can have more than one version of Eclipse running on your system at a time without a problem. If you previously installed a different version of Eclipse, this will add a new one with its own workspace, and you can always choose which one to run.
    4. The next screen will give you links to a set of Android Tutorials. Our class will not work with Android Apps at first. We will cover that material somewhat later. So you can skip those tutorials and just close the "Android IDE" tab.
    5. Now Eclipse is installed and ready to run Standard Java projects and Android projects.
    6. Note: Initially in this class, you will only be creating standard Java projects.
